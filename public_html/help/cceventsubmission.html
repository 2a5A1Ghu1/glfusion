Event Submissions

This is the command and control quick moderation area for calendar event submissions.  In this area you can edit, approve or delete event submissions.  To edit an event, click on the Edit icon. To approve or delete event submissions, select one  or more checkboxes and click on the appropriate action icon.

Field  Description
Edit Click to view or edit the event
Title  Title of the event
Start Start date of the event
URL URL for the event
