User Submissions

This is the command and control quick moderation area for user submissions.  In this area you can edit, approve or delete user registrations.  To edit a user, click on the Edit icon. To approve or delete user submissions, select one  or more checkboxes and click on the appropriate action icon.

Field  Description
Edit Click to view or edit the user.
Username  The user's selected username
Full Name The user's full name
eMail The user's e-Mail address
