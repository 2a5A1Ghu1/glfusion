To enter a backslash (\) in your post, just enter the \ \
[file]1[/file]

If file attachments are enabled, and you upload an image, you can include a thumbnail of the image in the post by using the [file]#[/file] tags.  The # is the number of the upload (1 is the first upload, 2 the second...).

This is will a thumbnail of the uploaded image into the post with a link to the original image. If you would like the original image to open in a 'lightbox' use the following syntax:

[file lightbox=1]#[/file]
