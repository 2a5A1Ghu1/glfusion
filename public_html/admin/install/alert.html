Important Upgrade Information - Please read the notice below...

Remove Old Files

With glFusion v1.6.0, there has been a significant restructuring of files in the CKEditor plugin and in the CMS theme. Please read the information below and ensure you remove the files that are no longer needed.

CKEditor FileManager

We have replaced the image browser, FileManager, in CKEditor with FileMan. As a result, you no longer need the FileManager files. Please remove the following directory (and all files included in this directory):

public_html/ckeditor/plugins/filemanager/

It is important to remove these as they will no longer be updated.

CMS Theme File Structure

With glFusion v1.6.0, we have moved all the plugin templates files from the CMS theme directory, back to the plugin's theme directory in the private/plugins/ space. It is important that you remove the old custom plugins files in the CMS theme directory.

CAUTION: If you have updated or customized any of these templates, do not delete your customized files.

The files to delete are located in the following directory: 

public_html/layout/cms/plugins/

It is safe to delete all files in this directory, but please leave the plugins/ directory - it should be empty (unless you leave any of your customized templates). Going forward, if you wish to customize a plugin's templates, you can store them in public_html/layout/cms/plugins/PLUGINNAME/templatefile.thtml - in the future, this directory will never be overwritten by glFusion, allowing your customizations to remain after upgrades.

For more detailed information on what files to remove, please see the glFusion Documentation Wiki - Upgrading to glFusion v1.6.x
