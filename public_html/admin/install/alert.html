Remove Old Files

glFusion v1.6.0 has re-organized many files. Please remove the following files / directories:

  • public_html/ckeditor/plugins/filemanager/* - We have replaced the CKEditor FileManager Plugin with a new file manager called FileMan. Please completely remove the filemanager directory.
  • public_html/layout/cms/plugins/ - glFusion v1.6.0 has moved all the custom plugin template files to the plugin's template directory - the old template files need to be deleted. Please delete all files and directories under the public_html/layout/cms/plugins/ directory (keep the plugins directory for future use).
