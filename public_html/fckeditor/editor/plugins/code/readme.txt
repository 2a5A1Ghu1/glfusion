v1.0 - Initial Release

Overview

CodeEmbed is a plugin for the FCKeditor that is included with glFusion. This
plugin will allow you to easily insert formatted source code into your
forum posts. Once installed, simply press the Code button on your editor
toolbar to open Code Embed. Paste your source code, and then press INSERT.
The source will be placed in the editor window.
